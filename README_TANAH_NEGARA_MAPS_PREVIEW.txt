Perbaikan Tanah Negara - Latitude, Longitude, dan Preview Google Maps

Perubahan:
1. Form Tanah Negara tetap menampilkan field latitude dan longitude.
2. Form tambah/edit Tanah Negara sekarang menampilkan preview Google Maps otomatis setelah latitude dan longitude diisi.
3. Halaman detail Tanah Negara menampilkan preview Google Maps jika koordinat tersedia.
4. Tersedia tombol Buka di Google Maps pada form dan detail.

Catatan:
- Preview menggunakan URL embed Google Maps berbasis koordinat, tanpa API key.
- Format koordinat bisa menggunakan titik desimal, contoh latitude -6.20000000 dan longitude 106.81666667.
- Jika koordinat kosong, sistem menampilkan pesan bahwa preview peta belum tersedia.
